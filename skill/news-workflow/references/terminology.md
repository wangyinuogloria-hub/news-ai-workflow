# 术语库

## 一、经济/产业高频术语

### 外贸领域
| 术语 | 用法说明 |
|------|---------|
| 出海 | 产品出口的形象化表述，需加引号 |
| "新三样" | 锂电池、光伏、新能源汽车，必须加引号 |
| 远销 | 描述出口范围，"远销欧美、东盟等全球多地" |
| 枢纽 | "全球跨境电商枢纽""外贸枢纽优势" |
| 全链条/全链路 | 描述产业完整性 |
| 提质增量 | 比"提质增效"更偏外贸 |
| "快车道" | 通关便利化的形象说法，需加引号 |
| 海外仓 | 跨境电商专用术语 |
| POD（按需定制） | 首次出现需加中文注释 |
| "抵港直装" | 海关便利化措施，需加引号 |
| "即买即退" | 离境退税措施，需加引号 |
| 保税维修 | 加工贸易术语 |

### 科技/产业领域
| 术语 | 用法说明 |
|------|---------|
| 产学研 | 不展开写"产学研用"，正文常用"产学研深度融合" |
| 新质生产力 | 习近平经济思想核心概念，需准确使用 |
| 固态电池/钠离子电池 | 前沿技术，需写全称 |
| 能量密度 | 电池技术指标 |
| AGV/AMR | 移动机器人类型，首次出现需括号注释 |
| 亚毫米级 | 精度描述 |
| "专精特新" | 中小企业资质，必须加引号 |
| 概念验证/中试熟化 | 科技成果转化环节 |
| 脑机接口 | 前沿技术，无需加引号 |

### 绿色/低碳领域
| 术语 | 用法说明 |
|------|---------|
| 近零能耗建筑 | 标准术语，不写"零能耗" |
| 近零碳排放建筑 | 标准术语 |
| 再生能源利用率 | 不写"可再生能源利用率" |
| 综合节能率 | 百分比表达 |
| "厂网一体" | 污水管网建设模式，需加引号 |

## 三、机构/政策标准表述

### 深圳海关系统
- "深圳海关"（非"深圳市海关"）
- "深圳海关所属布吉海关"（层级表达规范）
- "深圳海关所属大铲湾海关" 
- "深圳海关所属莲塘海关"
- "深圳海关自贸区和特殊区域发展处"（完整处室名称）

### 边检系统
- "深圳边检总站"（非"深圳边防检查总站"）
- "深圳边检总站西九龙边检站"
- "深圳边检总站莲塘边检站"

### 政府部门
- "财政部、住房城乡建设部"（两部门并列，中间用顿号）
- "深圳市口岸办"（非"深圳市口岸办公室"）

### 其他机构
- "中国资源循环集团"（央企全称）
- "深圳市电池行业协会"
- "深圳市跨境电子商务协会"
- "二十一世纪教育研究院"
- "哈工大（深圳）"（高校简写规范）

## 五、官媒规范用语（通用）

### 必须使用全称的
- "一带一路"（不可写为"一代一路"）— **注意：中欧班列≠一带一路**，中欧班列是连接中国与欧洲/中亚的铁路通道，只覆盖欧亚大陆，不能用"共建'一带一路'"来指代中欧班列。正确表述："中欧班列（深圳）是大湾区货物'出海'的稳定畅通国际大通道""为高质量共建'一带一路'筑牢物流根基"（中欧班列为一带一路贡献力量，但本身不等于一带一路）
- "十五五"（指第十五个五年规划，加引号）
- APEC（大写，首次出现不强制写全称）
- AI（大写）

### 领导讲话/政策引用规范
- 引用政策文件：使用书名号《》
- "通知明确""通知指出"（文件引用引导语）
- 会议精神表述：使用"..."围绕...这一主题

### 高频官媒句式
- "依托...优势" 
- "持续...，不断..."
- "进一步...，为..."
- "加快...，推动..."
- "助力...，促进..."

## 六、禁用/慎用词汇

| 禁用/慎用 | 替代方案 |
|-----------|---------|
| "牛逼""厉害了我的..." | 不使用网络流行语 |
| "疯狂""爆炸式"（过度夸张） | "快速增长""大幅提升" |
| "深圳速度"（已被过度使用） | 用具体数据替代 |
| 企业名称简写（"华为"除外） | 使用工商注册全称 |
| "老外" | "外籍旅客""海外客户" |
| "砍单""爆单"（电商俚语） | "订单增长""订单量提升" |

## 七、数据表达规范

- 大数字：超XXX亿元 / 达XXX亿元 / 突破XXX亿元
- 增长：同比增长XX% / 增长近X成 / 增幅达XX%
- 占比：占全国XX% / 占比超X成 / 贡献度超X成
- 范围：XXX～XXX / 超过XXX / 近XXX
- 时间窗口：前X个月 / X月份以来 / X月当月 / 一季度
