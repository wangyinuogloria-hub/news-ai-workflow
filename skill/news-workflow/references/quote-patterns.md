# 同期声选择模式

> 当前分析基于终稿中已选用的同期声特征。完整的选择/放弃规律需等配对样本（原始同期声 → 稿件同期声）补充。

## 一、同期声的功能分类

### 类型 1：数据支撑型（最常用，约占40%）
功能：用受访者之口说出关键数据，增加可信度

特征：
- 包含具体数字（金额、增速、数量）
- 通常来自企业高管或海关官员
- 数据有新闻价值（创新高、大幅增长、突破）

典型例子：
> "2026年我们前4个月的出口总量现在已经是7.5亿元，同比我们增长了将近150%。"（企业负责人）
> "2026年1-4月，经莲塘口岸出口的各类单独列名的机器人合计约7000台，同比增长约1122.9%。"（海关科长）

### 类型 2：政策/措施解读型（约占25%）
功能：解释某项政策或便利化措施的具体内容和效果

特征：
- 来自政府官员/海关人员
- 包含措施名称（加引号）+ 效果描述
- 语言规范，有公文风格

典型例子：
> "综合运用'抵港直装''7X24小时预约查验'等便利化措施，大幅压缩货物在港通关时间。"（海关官员）
> "灵活启用'潮汐通道'，确保高峰时段'通道全开、即到即检'。"（边检副队长）

### 类型 3：趋势/观点定调型（约占20%）
功能：由专家或协会负责人对行业趋势做判断，为稿件定调

特征：
- 来自学者/行业协会负责人/研究院专家
- 宏观视角，站得高
- 通常放在稿件靠后位置做总结升华

典型例子：
> "深圳走出了产学研融合的高阶形态。深圳不是简单的校企合作，而是形成了...产业—教育共同体。"（教育研究院专家）
> "借着APEC的东风，主动抢订单、调结构，防风险，培育新产品，稳住我们外贸的基本盘。"（哈工大深圳研究院院长）

### 类型 4：场景/体验型（约占10%）
功能：用一线亲历者的视角增加画面感和人情味

特征：
- 来自普通旅客/学生/消费者
- 语言口语化，有真实感
- 短小精悍（1-3句）

典型例子：
> "我就感觉这个车快了一点，然后我的眼睛就和它一起转的，挺神奇的，感觉是AI那种很科技的力量。"（三年级学生）
> "很快很方便，这个位置，也期待更多口岸开的。"（香港居民）

### 类型 5：记者出镜对话（约占5%）
功能：记者出现在镜头前，通过与一线人员的简短对话建立在场感

特征：
- **标签可为【同期声】或【现场声】**，实践中记者提问+受访者回答各一个【同期声】标签也很常见
- 记者提问要简短，受访者回答要有具体画面或数据
- 建立在场感，不喧宾夺主

典型例子：
> 【同期声】记者 地方电视台新闻栏目记者：现在这个客户就是在询问中欧班列发车的情况，是吧？
> 【同期声】李龙 深圳皓鹏国际物流供应链管理有限公司 销售代表：是的，走德国的客户吹风机发到德国，盘量大概在10吨左右，上个月就走了有100吨左右。

**使用原则（✅ 实战验证）：**
- 记者对话**服务于段落叙事**，不是独立的"表演"，必须嵌入场景正文之后，作为场景的延续
- ❌ 反面：把记者对话孤零零放在开头当"钩子" —— 突兀，不服务主旨
- ✅ 正面：正文描写福永片区业务员忙碌 → 记者指向一个正在发生的真实询盘 → 李龙回答，形成"场景→实证"的递进

## 二、同期声的选择标准（推测）

基于终稿中间接推断的选择偏好：

### 大概率被选用的同期声
1. **有具体数据的**："增长了150%""合计约7000台"
2. **能解释政策的**：海关便利化措施的具体操作
3. **有画面感的**：描述生产线/码头/活动现场的细节
4. **能定调升华的**：专家从宏观角度的判断
5. **有情感温度的**：学生惊叹、旅客点赞

### 大概率被放弃的同期声（推测，待配对样本验证）
1. 空洞的套话（"很有意义""非常好"但没有具体内容）
2. 与导语/正文信息重复的内容
3. 过于冗长、条理不清的即兴发言
4. 过于技术细节、观众难以理解的内容
5. 与新闻角度关联度低的内容

## 三、同期声在稿件中的排布逻辑

### 模式 A：递进式（经济/产业类常用）
```
同期声1（企业）：经营现状 + 同期声2（海关）：政策支持 + 同期声3（协会/专家）：行业展望
从微观 → 中观 → 宏观，逐层递进
```

### 模式 B：并列式（论坛/文化活动类常用）
```
同期声1（专家A）：观点A + 同期声2（专家B）：观点B + 同期声3（专家C）：观点C
同主题不同视角并列，互相补充
```

### 模式 C：对比式（口岸/消费类常用）
```
同期声1（旅客/消费者）：体验感受 + 同期声2（官员）：政策解读 + 同期声3（商户/企业）：经营数据
供需双方的视角形成对照
```

## 四、单篇同期声数量统计

| 稿件类型 | 典型同期声数量 | 范围 |
|---------|--------------|------|
| 简约动态（如通关消息） | 1 个 | 1 |
| 标准动态 | 2-3 个 | 2-4 |
| 主题报道（单篇） | 4-5 个 | 3-6 |
| 论坛/峰会综合稿 | 每子标题 2-5 个 | 总计 8-15 个 |

## 五、同期声拼接规律（✅ 已通过配对样本验证）

### 5.1 拼接特征
- 同一受访者的同期声可以跨段落出现（不同的正文段中引用同一个人）
- 拼接后同期声内部逻辑连贯，不跳话题
- 口头禅和重复词已被清理（如"然后""就是说""那个""对""好"）
- 保留受访者的语言风格（专业术语、口语化表达）

### 5.2 记者的现场引导（从配对样本中发现）
- 记者采访中多次说"你把这个话再说一下"——引导受访者用更精炼的语言复述
- 记者会把想要的句子框架"喂"给受访者："你看能不能总结一些话，提炼出来一些观点"
- 外国受访者（中文不流利）需要更大量的现场引导和后期拼接
- 记者的现场判断已经包含了"这段能不能用"的预筛选

## 六、同期声裁剪六条铁律（✅ 基于配对样本验证）

| # | 铁律 | 案例 |
|---|------|------|
| **1. 角度决定取舍** | 不符合角度的内容，说得再好也删。李玉成10分钟技术讲解（浸没式冷却、高倍率电芯、钠电池）因角度是"进出口规模"全部被废 |
| **2. 正文与同期声分工** | 背景/政策/数据由正文配音交代，同期声负责"亲口证实"。马约翰的次要品类数据（登机廊桥、建材）转为正文配音 |
| **3. 一人在一篇中只保留1-2句精华** | 原始采访10分钟+，终稿同期声只保留最精华的70-120字 |
| **4. 技术细节全部删除** | 观众不需要知道"浸没式集成技术""1C到8C倍率""钠电池零下40度充放电" |
| **5. 数据对比最受欢迎** | "30柜vs5柜""增长超两成""缩短5-7天"——有对比的数字天然适合电视同期声 |
| **6. 外国受访者需"预加工"** | 记者在现场把想要的句子框架给受访者，受访者确认复述，后期再拼接 |

## 七、正文配音 vs 同期声的分工（✅ 新增）

| 信息类型 | 放在正文（配音） | 放在同期声 |
|---------|----------------|-----------|
| 政策名称/背景 | ✅ 正文交代"海关推行的加工贸易企业集团监管模式" | ❌ 同期声不需要再解释政策 |
| 宏观数据 | ✅ 正文："进出口规模达1.82万亿元" | ❌ |
| 个体数据 | ❌ | ✅ "我们前4个月出口7.5亿元" |
| 真实感受 | ❌ | ✅ "挺神奇的，感觉是AI那种很科技的力量" |
| 措施效果 | ❌ | ✅ "极大的减少了我们缺料待产的这种情况" |
| 趋势判断 | 可正文可同期声 | ✅ 专家同期声更有分量 |
| 场景描写 | ✅ 正文："一辆辆满载机器人的集装箱货车..." | ❌ |

## 九、同期声筛选优先级（✅ 实战验证）

**读转录时按以下优先级标注，从高到低：**

| 优先级 | 类型 | 特征 | 案例 |
|--------|------|------|------|
| 🔴 P0 | **独家画面/活动名** | 有名字的活动、有日期的现场、独一无二的细节 | "夏日狂想曲""笔架山远足""必胜客披萨+定制可乐" |
| 🔴 P0 | **情感表达** | 有温度、有感染力、能让观众记住的话 | "唱唱跳跳、吼一吼""不要太紧张，不要太害怕" |
| 🟡 P1 | **数据对比** | 有具体数字的今昔对比、增幅 | "30柜vs5柜""增长超两成" |
| 🟢 P2 | **政策措施名** | 有名称的制度和做法 | "阳光体育""4-4-6呼吸法""全程监装+检验" |
| ⚪ P3 | **系统归纳** | 概括性陈述、趋势判断 | "三年日常化""心理越来越强大" |

**核心原则：P3最容易被正文替代，优先度最低。如果同期声只是"让受访者把正文能写的事实重说一遍"，就选错了。同期声的价值在于给出正文给不了的东西。**

## 十、"漏掉金句"自检（✅ 实战教训）

写完同期声框架后，回转录原文逐一检查：

1. 受访者有没有提到**有名字的活动**？→ 我选了吗？
2. 受访者有没有**有情感的表达**（哭/笑/激动/感叹）？→ 我选了吗？
3. 受访者有没有**独特的画面描述**（唱唱跳跳、吼一吼、赶出去）？→ 我选了吗？
4. 有没有哪句话**正文写不出来**但受访者说得很好的？→ 我选了吗？

**实战案例：** 田洪波采访中提到"昨天（6月1日）晚上夜场进行了夏日狂想曲的活动，把学生都赶到操场，让学生在音乐中放松身心，可以唱唱跳跳、吼一吼，能够宣泄心中的压力"。这句话同时满足P0的1和2（有名字的活动+情感表达），是教科书级的同期声素材。AI漏掉了，导师选中了。

## 十一、重复企业/机构的差异化逻辑（✅ 实战教训）

当一篇稿件中出现**两个以上同类企业/机构**受访者时，必须主动寻找区分逻辑，不能等用户指出：

### 寻找差异化的方法
1. **对比同期声侧重点**：一个偏市场/热度/规模，另一个偏模式/创新/趋势
2. **对比角色定位**：销售经理→市场一线感知；产品运营负责人→服务方案设计
3. **对比独有信息**：各自说了什么对方没说的事？独家数据、独家做法、独家方向
4. **功能分工**：一个证明"有多热"，一个证明"往哪走"

**实战案例：** 皓鹏（销售经理，偏市场热度）vs 大洋（产品运营，偏模式创新/过境货物）。如果AI自己没找到这个分工，稿件会变成两段重复堆砌。

### 常见分工模式
| 企业A | 企业B | 适用场景 |
|------|------|---------|
| 热度/规模 | 模式/创新 | 同类企业并列 |
| 深圳本土视角 | 国际化/出海视角 | 内外互补 |
| 当下数据 | 未来布局 | 时间递进 |
| 物流/货代 | 制造/卖家 | 供需两端 |

## 十二、不要漏人（✅ 实战教训）

**转录文本中的每一个人都要过一遍**，即使主受访者说了90%的内容：
- 开头简短出镜的人 → 可能是一段有画面感的同期声
- 记者与次要人物的对话 → 有在场感、有画面
- "这个人没说几句" ≠ 没用，恰恰可能是稿件中最鲜活的素材

**实战案例：** 皓鹏转录中有李龙（销售代表），记者问他客户询盘情况，他回答"一天询盘量10吨，中午半天走了100吨"。这段只有20秒但画面感极强。终稿把记者提问+李龙回答各自标为【同期声】，嵌入福永片区场景正文之后，形成了"场景→实证"的递进。

## 十三、同期声裁剪终极原则（✅ 实战验证）

**同期声只保留"正文写不出来的东西"。** 凡是正文能交代的背景、数据、解释，一律移到正文配音。

| 信息类型 | 放哪 | 案例 |
|---------|------|------|
| 海外仓城市清单、面积 | **正文** | "企业投资建立的海外仓已覆盖意大利、比利时、匈牙利等多个国家，总面积超6万平方米" |
| 时效缩短具体天数 | **正文** | "海外客户货物派送时间大幅缩短" |
| 门到门25-26天 | **正文** | ← 正文一句话就够了 |
| 过境货物新模式 | **同期声** | "依托深圳中欧班列发运，提高东南亚客户出口到欧洲跟莫斯科的连接" |

**实战教训：** 黄泽彬采访中，海外仓布局细节（6万㎡、5个国家城市）全部移到了正文，同期声**只保留了过境货物一个点**。同期声的价值不在信息量，在"这句话只有这个人能说"。

## 十四、拼接操作手册（给AI的指令）

### 从原始同期声到终稿同期声的操作步骤：

1. **通读原始转录**，按第九章优先级标注所有可用的信息片段
2. **P0优先**：先锁定有名字的活动和有情感的表达，这些是不可替代的同期声素材
3. **根据稿件角度筛选**：只保留与角度直接相关的信息点
4. **分离正文素材和同期声素材**：P3（系统归纳）交给正文，P0/P1留给同期声。**关键判据：这件事正文能不能写？能写就交给正文。同期声只留"正文写不出来的"**
5. **跨段落拼接**：从转录全文各处摘取最精华的碎片，压缩到60-90字（20秒以内）
6. **清理噪点**：删除口头禅、重复、与角度无关的内容、过于技术的细节
7. **保持原味**：保留受访者的口语感染力和情感表达，不要"书面化"
8. **检查伦理底线**：不可修改措辞，不可歪曲原意，不可无中生有

## 十五、P0判断陷阱（✅ 2026-06-11 实战验证）

**不是所有"精彩的话"都是P0。** 抽象洞察即使听上去有深度，也不一定适合做同期声。

### P0误判案例（AI选中，终稿砍掉）

| AI选的P0 | 终稿替换成 | 为什么AI选错了 |
|-----------|-----------|---------------|
| "硬件能力非常强了，但大脑还是有所欠缺" | "深圳供应链成熟，做人形机器人速度非常快，需要算法人员在机器人大脑控制上突破" | 前者是技术判断（静态），后者是深圳优势+产业需求（动态叙事） |
| "很多前沿技术还在实验室，企业需要完成从零到一的跨越" | "把3C的产品质量和定制化设计理念延展到具身智能" | 前者是抽象呼吁，后者是可操作的产业路径——观众能听懂3C，能想象"延展" |
| "卷技术卷价值，不是生产更便宜的设备" | "一台设备能干原来1.2到1.6台的设备" | 前者是态度表态，后者是具体对比数字——电视观众记住数字不记住态度 |
| "AI工艺大师把20年老师傅经验数字化" | 效率提升20%-60%的数据 | 前者是产品名+概念，后者是可感知的效果 |

### 修正后的P0判定标准

| 真P0（同期声） | 假P0（容易误判，应交给正文或放弃） |
|---|---|
| 有名字的活动、有日期的现场 | 抽象的行业判断（"从A转向B"） |
| 有情感的表达（哭/笑/激动） | 有洞察但无画面的评论（"零到一的跨越"） |
| 有画面的具体描述（"拖拉拽""搭积木"） | 有态度但无数据的表态（"卷技术卷价值"） |
| 可感知的数字对比（"1台干1.2~1.6台"） | 有名称但无解释的产品概念（"AI工艺大师"） |
| 可操作的产业路径（"把3C经验延展到具身智能"） | 呼吁式判断（"需要尽快走向工业现场"） |

**核心判据修正：**
旧标准："这句话正文能不能写？能写就交给正文。"
新标准："观众听完这句话，脑子里多了一个可感知的信息，还是多了一个抽象判断？前者选，后者砍。"

### 同期声口语保留原则（新增）

清理口头禅时区分两类：
- **删掉**：无意义重复（"就是说""那个那个""然后然后"）
- **保留**：语感词（"应该""大概""你会发现""平均下来"）、口语连接词（"所以""跟我们...对比"）、真实的语序（"大概接触了有10-20个这个客户"）

**判据：** 删掉这个词后，同期声还像不像一个真人在对着镜头说话？不像就保留。

### 同期声口语"广播适格"原则（✅ 2026-06-28 实战验证）

不是所有口语都适合电视播出。在"保留真人感"和"符合新闻播出标准"之间需要做区分：

**删掉（不符合广播适格）：**
- 情绪感叹词："好爽啊""太爽了""哇塞""牛逼"
- 口语招呼词（无信息量）："你看这个""我跟你说啊""你知道吗"
- 过度随意的口语用词：视上下文判断，同期声听起来是否像在聊天而非在接受采访

**保留（真人感 + 广播适格兼具）：**
- 语感词："应该""大概""你会发现""平均下来"
- 口语连接词："所以""跟我们对比"
- 真实语序和轻微重复

**核心判据修正：**
旧判据："删掉后还像真人在说话吗？不像就保留。"
新判据："删掉后还像真人在说话吗？不像就保留。**同时：保留后像新闻同期声吗？不像就删。**"

**实战案例：** 姚梓骏（香港市民）原文"你看这个场地，这么新有八片场地，好爽啊"→ 删"你看"（口语招呼词）+ 删"好爽啊"（情绪感叹词），保留"这个场地这么新，有八片场地"（有画面、有信息量、播出得体）。
